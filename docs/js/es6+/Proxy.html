# Proxy

Proxy用于拦截修改目标对象某些行为的默认操作，类似 ES5 中Object.defineProperty的效果，但可拦截的内容更多，不止get/set操作，且Proxy是对返回的代理对象操作才会触发拦截，不像Object.defineProperty是直接操作原目标对象触发拦截。

语法：

// 参数1 target 为要拦截的目标对象（可以是任何东西，包括函数）
// 参数2 handler 为要拦截的具体操作配置对象，以拦截的行为和对应的处理函数组成，若 handler 是个空对象（即没有任何拦截），则对 proxy 和 target 的操作是同步的（但仍不是同一个对象）
// 返回一个 Proxy 实例(一个新代理对象)，后续所有操作需要在该新对象上进行才会触发拦截操作
const proxy = new Proxy(target, handler); // 后面提到的`proxy`都表示一个实例对象

WARNING

  • 要使得Proxy起作用，必须针对返回的Proxy实例proxy对象进行操作，而不是对原目标对象target进行操作；
  • 对于拦截配置对象handler中可以设置但未设置拦截的操作，则默认落在原目标对象target上，按照原先的方式产生结果；
  • 若拦截配置对象handler是个空对象（即没有任何拦截），那么对proxy和target的所有操作都是同步的，但仍是两个不同的对象；

proxy 对象可以设置到其他对象的属性上，也可以作为其他对象的原型对象。

const obj1 = { proxy: proxy };  // 放在其他对象属性中
const obj2 = Object.create(proxy);  // 作为其他对象的原型对象

Proxy 支持拦截的操作有如下 13 种：

get、set、has、deleteProperty、ownKeys、getOwnPropertyDescriptor、defineProperty、preventExtensions、getPrototypeOf、isExtensible、setPrototypeOf、apply、construct

以下拦截处理器中形参说明：

  • target，原始目标对象（apply和construct中为函数）
  • propKey，目标属性(键)名
  • receiver，当前 proxy 实例对象
  • value，设置的值

具体作用：

  • get(target, propKey, receiver) 拦截对象下的属性访问，包括没有的属性
  • set(target, propKey, value, receiver) 拦截对象下的属性的设置，包括新增属性，返回一个布尔值
  • has(target, propKey, value, receiver) 拦截 propKey in proxy 操作，返回一个布尔值
  • deleteProperty(target, propKey) 拦截 delete proxy[propKey] 操作，返回一个布尔值
  • ownKeys(target) 拦截以下操作，返回一个数组。该方法返回目标对象所有自身属性的属性名，而 Object.keys/values() 的返回结果仅包括目标对象自身的可遍历且非 symbol 属性/值
    • Object.getOwnPropertyNames(proxy)，返回非 symbol 键
    • Object.getOwnPropertySymbols(proxy)，返回 symbol 键
    • Object.keys(proxy)，返回目标对象自身的可遍历且非 symbol 键
    • Object.values(proxy)，返回目标对象自身的可遍历且非 symbol 值
    • Object.entries(proxy)
    • for...in 循环，遍历所有带有 enumerable 标志(即可遍历)的非 symbol 键，以及原型对象的键
  • getOwnPropertyDescriptor(target, propKey) 拦截 Object.getOwnPropertyDescriptor(proxy, propKey)，返回属性的描述对象
  • defineProperty(target, propKey, propDesc) 拦截以下操作，返回一个布尔值
    • Object.defineProperty(proxy, propKey, propDesc)
    • Object.defineProperties(proxy, propDescs)
  • getPrototypeOf(target) 拦截 Object.getPrototypeOf(proxy)，返回一个对象
  • setPrototypeOf(target, proto) 拦截 Object.setPrototypeOf(proxy, proto)，返回一个布尔值
  • preventExtensions(target) 拦截 Object.preventExtensions(proxy)，返回一个布尔值
  • isExtensible(target) 拦截 Object.isExtensible(proxy)，返回一个布尔值
  • apply(target, object, args) 拦截 Proxy 实例作为函数调用的操作（此时target 应该是一个函数），比如 proxy(...args)、proxy.call(object, ...args)、proxy.apply(...)
  • construct(target, args) 拦截 Proxy 实例作为构造函数调用的操作（此时target 应该是一个构造函数），比如 new proxy(...args)
const obj = {};
const proxy = new Proxy(obj, {
  get(target, propKey) {
    // 参数1 target 为代理的原目标对象
    // 参数2 propKey 为访问的属性名
    return '拦截处理值'
  }
});

WARNING

代理应该在所有地方都完全替代目标对象，目标对象被代理后，任何人都不应该再引用目标对象，否则很容易出意料之外的问题，可以使用如下变量覆盖方式来避免该问题：

let obj = {};
// 覆盖原目标对象，使得后续访问的 obj 变量实际都是代理后的对象
obj = new Proxy(obj, ...);
