# 数组Array

同对象一样，在使用数组字面量[]表示法时，也不会调用Array构造函数(即不会隐式通过new的方式调用)，另外，在使用Array构造函数创建数组时也可以省略new操作符（而Object中不能省略）

# 创建与访问数组

使用Array构造函数创建数组传入一个参数时，根据这个参数类型的不同将有两种结果

  • 传入的是一个数字则代表数组长度，得到一个该长度的数组
  • 传入的是其他类型参数，则得到的是包含该数据的只有一项的数组

数字索引也可以是字符串形式'0'，arr[0]===arr['0']，数组就相当于特殊形式的对象。

数组Array属于引用类型，在传递内容给其他变量时，不是复制数组内容，只是传递数组数据在内存中的地址指向。

数组是一个连续的储存空间，中间未赋值或未定义的单元格其值默认是undefined，在打印时内容是空的。

在数组已存好数据后，再通过length属性设置数组长度时，若设置的长度比数组的实际长度小，则数组超出设定长度部分的数据将会被删除；但若先指定数组长度，在实际存储数据时超过此长度，数组将会自动增加长度。

const arr = new Array(); // 空数组 []
const arr = new Array(7); // 长度为 7 的空数组 [,,,,,,]
const arr = new Array('7'); // 内容为 '7' 的一个长度的数组 ['7']
const arr = new Array(1, 2, 3); // 内容为 1,2,3 的三个长度的数组 [1, 2, 3]

const arr = []; // 字面量方式创建数组

# 数组类型判断

const arr = [];

typeof arr; // object
arr instanceof Array; // true
arr.constructor === Array; // true

Array.isArray(arr); // true，返回布尔值，用于判断 arr 是否是一个真正的数组，IE9+支持

# 首尾增删数组内容

// 在数组尾部增加内容，多个内容用逗号隔开，长度也会相应地增加
arr.push(...args);  // 返回数组长度

// 删除数组最后一个位置内容，长度也会相应地减少一个
arr.pop();  // 无参数，返回删除的数据

// 在数组头部增加内容，多个内容用逗号隔开，长度也会相应地增加
arr.unshift(...args); // 返回数组长度

// 删除数组第一个位置内容，长度也会相应地减少一个
arr.shift(); // 无参数，返回删除的数据

# 任意位置增删数组内容

使用splice()方法可在数组任意指定位置删除或插入内容，该方法会改变原数组，且始终会以数组形式返回被删除的项（没有删除项时则返回空数组）。

// 删除的同时插入新内容，返回删除的部分
arr.splice(start, num, ...val); // 参数：start开始删除位置下标(包含)、num删除单元格个数、val新增数据（多个数据则往后依次逗号隔开添加）

// 只删除数组内容，返回删除的部分
arr.splice(start, num); // 不传入第三个参数即可

// 只插入新内容，未删除则返回空数组[]
arr.splice(start, 0, ...val); // 将第二个参数---删除个数 num 设为 0 即可

因为数组也属于对象，所以用delete运算符也可删除数组中的数据，如delete arr[index]，但这样删除不同于pop、shift、splice，delete只会清除对应下标中的数据，而这个单元格实际还是存在，只是值变成默认undefined了。

# 截取数组

// 返回截取部分的数组或字符串，此方法不会改变原字符串或原数组
arr.slice(start, end);  // 参数：start开始截取位置、end结束位置（不包括，省略则默认为数组长度，若 start 也省略则会默认为0，此时就相当于浅拷贝了一个原数组）
//参数可为负值，此时该负数会自动加上数组长度来确定位置，注意只有负参数加，正参数不加，并不是两个参数都要加(可将负数理解为倒序查找)，同样不包括结束位置，end 比 start 小时则返回空数组

[].slice.apply(arguments); //可用于将函数参数伪数组转换为标准数组，也可以用ES6新增的 Array.form() 方法

# 数组拼接

// 字符串或数组拼接，返回拼接后的字符串或数组，此方法不会改变原字符串或数组
// 可拼接多个，参数可以是数组，也可以是展开的单个数据，也可两者混合使用，数组会自动被展开后拼接（只会展开一层）；
// 数组与字符串可混合拼接，结果数据类型以 arr1 为准，字符串后拼接数组时，[]会去除，但分隔数据的逗号`,`会以字符串形式保留下来
arr1.concat(...arr2);
[1, 2].concat([3, 4], 5, [6, 7, [8]]); // [1, 2, 3, 4, 5, 6, 7, [8]]

//字符串后拼接数组时，对象元素会被显示成[object Object]，因为是字符串，所以里面的数据是无法访问到的，在将数组用 join 方法转换为字符串形式时也会出现此情况

// join 方法将数组 arr 内容展开后以指定分隔符连接成一个字符串，此方法连接大量字符串片段时效率比用加号`+`更高
// 分隔符默认为逗号`,`，若要实现无间隔连接，可用空字符串(不是空格)作为分隔符：arr.join('')
arr.join('分隔符');
//调用数组的 toString() 方法也会返回由数组中每个值的字符串形式(null 和 undefined 将以空字符串表示)拼接而成的一个以逗号分隔的字符串，作用同 arr.join() 或 arr.join(',')，而调用 valueOf() 方法返回的还是原数组

# 调整数组排序

// reverse 将数组 arr 的内容倒序排列，并返回颠倒后的数组，注意此方法会改变原数组 
arr.reverse();

// sort 方法按给定的比较函数 comparefn 返回的结果对数组进行排序，此方法会改变原数组
// 若不指定 comparefn 函数，将默认按字符编码的先后顺序排列(无实用性)
arr.sort(comparefn);
//comparefn 包含两个参数(a, b)，表示相邻两个单元格的值（每次排序之后的），会从第一个单元格一次遍历调用，返回`负数`时则按[a,b]排列(a排在b前)，返回`正数`则按[b,a]排列(a排在b后)，返回`0`则顺序不变

# 数组内容查找

// 查找指定内容 sch 在 arr 中的索引位置，返回查找到的对应项的索引位置，未找到则返回 -1，IE9+支持
arr.indexOf(sch[, start]);  // 参数 sch 为查找的内容，可选参数 start 为起始查找位置(索引)

// 与上面 indexOf 作用一样，只是 lastIndexOf 是倒序查找(即从指定位置开始向前查找)，而 indexOf 是正序查找(即从指定位置开始向后查找)，IE9+支持
arr.lastIndexOf(sch[, start]);

// 以下为 ES6 新增，IE不支持

// find 用于找出第一个符合条件的数组成员，注意是第一个，没有则返回 undefined
// 依次对每个成员调用传入的回调函数，返回 true 则表示符合条件，可传入第二个可选参数thisObj，其用于绑定第一个回调函数中的 this 对象指向
arr.find((item:数组项, index:对应索引, arr:整个数组) => item === 'val'[, thisObj]);

// 同上面 find 方法，只是 findIndex 返回的是找到的数组成员的索引位置，没有则返回 -1
arr.findIndex((item:数组项, index:对应索引, arr:整个数组) => item === 'val'[, thisObj]);

// 另外 ES2022 新增了以下 findLast 和 findLastIndex 方法，作用同上面，只是其都为从最后一个开始倒序查找，由于较新，Chrome97+ 才支持
arr.findLast(fn[, thisObj]);
arr.findLastIndex(fn[, thisObj]);

# 数组遍历执行

数组的属性名是整数时才会影响其length属性，所以在给数组扩展非整数属性或方法时不会影响数组长度，for循环数组中的数据时也将不受影响，但for...in遍历会影响

// 以下遍历方法都不会改变原数组，IE9+支持
// 每个方法都可接受2个参数：1每个数组成员上调用的函数fn、2可选的 fn 中 this 绑定对象 thisObj
// 传入的函数 fn 会接收3个参数：1当前数组项的值item、2当前索引index、3数组本身arr

// 若每一项都返回true，则该方法结果返回true，否则返回false
arr.every((item, index, arr) => item === 'val'[, thisObj]);

// 只要有某一项返回了true，则该方法结果就返回true，若所有项都没返回true则该结果返回false
arr.some((item, index, arr) => item === 'val'[, thisObj]);
// every() 方法与 some() 方法就像 && 与 || 的关系

// 返回一个数组，该数组由函数返回true的项组成
arr.filter((item, index, arr) => item === 'val'[, thisObj]);

// 返回一个数组，该数组由函数fn的返回值组成
arr.map((item, index, arr) => item + 1 [, thisObj]);

// 遍历数组成员，不返回值，本质上与 for 循环数组一样
arr.forEach((item, index, arr) => 'do sth' [, thisObj])

forEach方法中无法使用break等方式跳出循环，要实现在其中达到条件跳出循环可采用抛出错误的方式来中断循环。

try {
  var arr = ['first', 'second', 'third', 'fourth'];
  // 执行到第3次，结束循环
  arr.forEach(item => {
      if (item === 'third') throw new Error('EndIterative');
      console.log(item);// first, sencond
  });
} catch(e) {
  if(e.message !== 'EndIterative') throw e;
};
// 下面的代码不影响继续执行
console.log('sth');

# 数组成员迭代

迭代指每个数组成员依次与前一个成员进行作用，直到作用到最后一个，返回最后的迭代结果值，注意不是更新每个数组成员的值后返回，而只是返回迭代到最后一个的值。

// reduce 和 reduceRight 方法都接受2个参数：1每个数组成员上运行的迭代函数fn、2可选的迭代初始值initVal(不传该值时就默认将起始项作为初始值，然后从下一项开始迭代)
// 迭代函数fn接受4个参数：1前一个结果值preVal、2当前值curItem、3当前索引index、4数组本身arr，该函数的任何返回值都会自动传给下一次迭代函数运行的第一个参数preVal

// 从数组第一项开始逐个迭代到最后一项，返回最后一个迭代结果，不传如第二个可选参数initVal时，默认将数组第一项作为初始值，从第二项开始迭代
arr.reduce((preVal, curItem, index, arr) => preVal + curItem [, initVal]);

// 与上面 reduce 方法一样，只是 reduceRight 的迭代顺序是从数组最后一项开始逐个迭代到第一项，返回最后一个迭代结果
arr.reduceRight((preVal, curItem, index, arr) => preVal + curItem [, initVal]);

# 使用扩展运算符

扩展运算符...(三个点)，可将数组(或对象或字符串)的第一层展开，注意展开后是几个独立个体，不是合并成了连续字符串。

console.log(...[🍎, 🍌, 🍐]); // 🍎 🍌 🍐

// 用于解构赋值，注意用于数组赋值时扩展运算只能放最后(接受前面匹配后剩下的所有数据)
const [a, ...rest] = [🍎, 🍌, 🍐, 🌰];
console.log(a); // 🍎
console.log(rest); // [🍌, 🍐, 🌰]

# 将伪数组等转为真数组

前面提到数组截取方法slice()可以将伪数组(类数组，本质为具有length属性的对象)转为真数组，而 ES6 新增的Array.from()静态方法的适用范围更广，其可以将伪数组(类数组)或具有遍历接口(iterable)的对象转化为真正的数组返回(扩展运算符也能做相同的事(对象转数组除外))，伪数组就像NodeList集合、函数arguments这种，具有遍历接口的就像字符串，也包括ES6中新增的Set和Map这种类型的数据。

// 参数 arrLike 为待转换的伪数组或具有遍历接口(iterable)的数据，若不是一个符合要求的数据，则会返回一个空数组
// 第二个函数参数 mapFn 可选，作用类似数组的 map 方法，将 arrLike 转换后再由 mapFn 方法遍历处理后返回，Array.from(arrLike, mapFn) 等同于 Array.from(arrLike).map(mapFn)
// 第三个参数 thisObj 可选，是一个绑定 mapFn 中 this 指向的对象
Array.from(arrLike[, mapFn, thisObj]);
Array.from({0: 'a', 1: 'b', length: 2}); // ['a', 'b']，若传入的参数本就为数组，则将返回一个一样的新数组
Array.from({0: 'a', 1: 'b'}); // []，无 length 属性不被当做伪数组，也无 iterable 接口，故返回空数组

// 以下等同于：Array.from({'0': 1, '1': 2, '2': 3, length: 3}).map(item => Math.pow(item, 2))
Array.from({'0': 1, '1': 2, '2': 3, length: 3}, item => Math.pow(item, 2));  // [1, 4, 9]

// 用于将字符串转换为数组再返回长度，其能正确处理各种 Unicode 字符，避免js将大于 \uFFFF 的 Unicode 字符算作两个字符长度的 bug
Array.from('str🍎').length; // 4
console.log('str🍎'.length); // 5
