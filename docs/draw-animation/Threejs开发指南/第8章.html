# 第8章 创建/加载高级网格和几何体

通过three.js的组合函数(成组)和THREE.Geometry.merge()合并函数来组建/合并多个几何体，或从外部加载复杂几何体和网格。

# 8.1 几何体的组合与合并

# 8.1.1 对象组合

使用THREE.Group创建组对象，其与THREE.Object3D非常接近，唯一不同的是它本身不含有任何可渲染的数据(几何体/材质等)。使用add()方法往组中添加组成员，组内的每个对象的位置、缩放和旋转参数将都是相对于所在组对象对应参数的，组内的每个对象仍然可以单独进行平移、缩放和旋转等操作，但对组进行相关操作时就是整个组相对该组的中心进行整体的平移、缩放和旋转等操作，而不是每个组成员独立进行(组成员间的状态不会相对改变)。

const group = new THREE.Group();
group.add(geomMesh1);
group.add(geomMesh2);
scene.add(group);

# 8.1.2 将多个网格合并成一个网格

使用组时，每个对象仍是独立的，仍需要对他们分别进行处理和渲染，当对象数量非常多时仍会出现性能瓶颈。通过THREE.Geometry.merge()方法，可以将多个几何体合并成一个联合体，此时就变成了一个对象，当然也就不能对原先的各几何体分别进行操作了，也不能像原来为每个几何体添加材质，因为现在就是整体一个几何体。

const geom = new THREE.Geometry();
const geomMtl = new THREE.MeshNormalMaterial({
  transparent: true,
  opacity: 0.5
});
// 假设geomMesh1、geomMesh2...是已知的几何体网格
geomMesh1.updateMatrix(); // 更新对象的变换矩阵，以确保能正确合并各子对象
geom.merge(geomMesh1.geometry, geomMesh1.matrix);
geomMesh2.updateMatrix();
geom.merge(geomMesh2.geometry, geomMesh2.matrix);

scene.add(new THREE.Mesh(geom, geomMtl));

# 8.2 从外部资源加载几何体

# 8.2.1 以three.js的JSON格式保存和加载

1、保存和加载THREE.Mesh网格对象(即单个物体)

直接调用网格对象的toJSON()方法即可返回该网格对象的json格式数据，使用THREE.ObjectLoader.parse()可将一个json格式的网格数据加载解析为网格对象，也可以使用THREE.ObjectLoader.load(url)直接加载解析一个独立的json文件得到网格对象。

const geom = new THREE.BoxGeometry(1, 1, 1);
const mtl = new THREE.MeshNormalMaterial();
const mesh = new THREE.Mesh(geom, mtl);
// 导出网格对象的JSON数据
const result = mesh.toJSON(); // 导出的json数据是对象形式，需要转为字符串形式才能便于存储
localStorage.setItem('meshJson', JSON.stringify(result));

// 加载JSON格式的网格数据
const loader = new THREE.ObjectLoader();
const mesh2 = loader.parse(jsonResult); // 若json数据是字符串形式则需要先用`JSON.parse()`转换为对象格式使用

2、保存和加载场景(即整个场景内容)

此处场景数据指的除了场景内所有网格对象物体外，还包括光源和摄像机。保存和加载场景数据的方法同上面单个物体一样，只是是在场景对象scene上调用toJSON()方法而已。
