# 3.1 Three.js中不同种类的光源

Three.js中有许多不同种类的光源，每种光源都有特别的行为的用法。

  • THREE.AmbientLight是环境光，是最基本的光源，该光源颜色会叠加到场景内现有物体的颜色上
  • THREE.SpotLight是聚光灯光源，就像聚光灯或手电筒，该光源可用于创建阴影
  • THREE.PointLight是点光源，就像夜空中的照明弹，由空间中一点向所有方向发射光线，该光源可用于创建阴影(低版本不行)
  • THREE.DirectionalLight是平行光光源，就像太阳光，该光源可用于创建阴影
  • THREE.HemisphereLight是半球光光源，其可以模拟反光面和光线微弱的天空来创建更加自然的室外光线(考虑的天空和地面的反射)，该光源不能用于创建阴影
  • THREE.AreaLight是区域光光源或平面光光源，其可以指定散发光线的平面，而不是一个点，就像亮着的显示屏，该光源不能用于创建阴影
  • THREE.LensFlare不是一种光源，但是使用该效果，可以为场景中的光源添加镜头光晕效果

其中环境光、聚光灯光源、点光源、平行光光源为基础光源，其他为特殊光源，这些光源都是基于THREE.Light对象扩展的，该对象提供了公用的功能，所有光源都继承了该基础类的属性和方法。

// new THREE.Light(color, intensity); color参数为颜色，可选intensity参数为强度，即决定该光线的亮度和对场景中物体颜色的影响程度，默认为1
var light = THREE.Light('#606008', 1); // 第二个参数光照强度可省略
light.color = new THREE.Color('#666'); // 重置光线颜色
light.intensity = 2; // 重置光的强度，值越大则光源越明亮，默认1
light.copy(otherLight); // copy()方法将其他光源的color和intensity复制到当前光源对象中
// 对于有方向的光源也可以使用 light.lookAt() 方法让光源朝向某方向(默认都是场景中心)，有的也可以使用 light.target 属性让光源指向某目标

# 3.2 基础光源

# 3.2.1 THREE.AmbientLight（环境光）

// new THREE.AmbientLight(color, intensity); color参数为颜色，可选intensity参数为强度，即决定该光线的亮度和对场景中物体颜色的影响程度，默认为1
var light = THREE.AmbientLight('#606008', 1);

该光源的颜色将会应用到全局(场景内的所有物体)，且其没有特别的来源方向，也不会生成阴影。一般不会将该光源作为场景中唯一的光源，因其会将场景中所有的物体渲染为相同的颜色。该光源与其他一同使用的目的是弱化阴影会给场景添加一些额外的颜色。

# THREE.Color

在Three.js中需要制定颜色时(例如材质、灯光等)，可以使用THREE.Color对象，也可以直接字符串指定一个颜色(一般是在使用相关类进行实例化时作为参数可以，当通过color等属性进行赋值时需要创建THREE.Color对象)，此时Three.js内部仍然会自动使用该字符串颜色去创建一个THREE.Color对象(也就是说我们获取到的所有颜色数据实际都会是一个THREE.Color对象)。

// 参数可以有很多形式
var color = new THREE.Color();  // 无参数时，会创建一个代表白色的颜色对象
color = new THREE.Color(0x666666); // 十六进制数值颜色值(最佳方式)
color = new THREE.Color('#666666'); // 十六进制CSS格式颜色值
color = new THREE.Color('rgb(21, 32, 44)'); // rgb格式颜色值，分量可以是1-255的整数，也可以是百分比值
color = new THREE.Color('yellow'); // 字面颜色名称值
color = new THREE.Color('hsl(0%, 100%, 50%)'); // hsl格式颜色值
color = new THREE.Color(1, 0, 0); // 分离的rgb值，此时三个值分别代表R、G、B三个分量，但其取值范围都是0~1，可看作是百分比

修改一个现有颜色对象的颜色，可以用新颜色值构造一个新的THREE.Color颜色对象进行覆盖赋值(就像上面那样)，也可以使用THREE.Color对象上的方法来读取和修改其内部颜色值。

// 使用THREE.Color对象上的方法获取和设置其内部颜色
color.set(newColor); // set()方法将当前颜色设置为指定的颜色，新颜色newColor是十六进制值，可以是字符串，也可以是一个THREE.Color实例
color.setStyle(styleColor); // 设置为指定的颜色，要求参数styleColor为css样式中的颜色格式(css中支持的都可以)
color.getStyle(); // 以CSS颜色格式(rgb格式)返回当前的颜色值
color.getHex(); // 以十六进制值格式返回当前的颜色值(实测其返回的是转换为十进制后的数值结果)
color.getHexString(); // 以十六进制字符串值格式(不带前面的'#'或'0x'标识)返回房钱的颜色值

# 3.2.2 THREE.SpotLight（聚光灯光源）

该光源较常用，是一种具有锥形效果的光源，其产生的光具有方向和角度，可形成阴影。

// 创建聚光灯光源只需要实例化时指明灯光颜色，然后设置需要的属性后添加进场景即可
// new THREE.SpotLight(color, intensity); color参数为颜色，可选intensity参数为强度
var spotLight = new THREE.SpotLight('#ffffff');
spotLight.position.set(-40, 60, -10);
spotLight.castShadow = true;
// 当指定了target时，光源将始终照向它(但并不是只照在该物体上，只是焦点在该物体上)，即使它在场景中移动。（默认应该是照向场景中心？）
spotLight.target = plane; // 让聚光灯照向地面(假设plane是代表地面的对象)

// 当不想将光源照向一个特定的物体，而是空间任意一点时，可以创建一个THREE.SpotLight对象给target
var target = new THREE.Object3D();
target.postion = new THREE.Vector3(5, 0, 0);
spotLight.target = target;

辅助类THREE.CameraHelper可用来展示场景中各种摄像机所覆盖的范围，聚光灯的辅助类THREE.SpotLightHelper可用来展示相关聚光灯在场景中作用的范围（轮廓）。

// 这里使用THREE.CameraHelper对象展示阴影作用范围(因阴影也是由光源的相机决定如何绘制的)
var cameraHelper = new THREE.CameraHelper(spotLight.shaow.camera);
scene.add(cameraHelper);
// THREE.SpotLightHelper辅助类可展示相关聚光灯在场景中作用的范围（轮廓）
var spotLightHelper = new THREE.SpotLightHelper(spotLight);
scene.add(spotLightHelper);
function render() {
  ...
  cameraHelper.update(); // 若阴影相机在变化，则也需要在render方法中同步调用`update()`方法更新cameraHelper视图
  spotLightHelper.update(); // 若聚光灯光源在变化，则也需要在render方法中同步调用`update()`方法更新SpotLightHelper视图
  ...
}

spotLight.penumbra属性可以设置聚光灯光锥边缘位置光线强度的衰减速度，取值0~1，默认0即不衰减，此时光照区域与未照射区域的分界就非常明显，设置大于0的值时边缘就会往内收缩形成一个过渡边缘区域，类似CSS中内阴影的效果。

关于阴影的几点注意事项：

  • 当阴影看着有点粗糙(如边缘锯齿严重)，则可增加shadow.mapSize.width和shadow.mapSize.height两个值的大小。或者保证阴影作用区域紧包围在对象周围，即通过设置shadow.camera对象上的near、far、fov三个属性来使阴影作用区域与对象更贴合。
  • 不仅要设置光源的castShadow让其可生成阴影，还必须配置每个对象的castShadow或receiveShadow属性来告诉对应物体是否投射阴影或者接受阴影(如地面)，默认都是不投射或接受阴影的。
  • 当场景中使用了薄对象(无厚度或厚度很小)，若其渲染阴影是出现奇怪的失真现象，可以通过shadow.bias属性修复。
  • 若要阴影更加柔和，可更改THREE.WebGLRenderer对象上shadowMap.type属性的值，其默认为THREE.PCFShadowMap，可设置为PCFSoftShadowMap显得更加柔和。

# 3.2.3 THREE.PointLight（点光源）

该光源是一种单点发光，照射所有方向的光源（如夜空中的照明弹），其可用于产生阴影(低版本不行)。

// new THREE.PointLight(color, intensity); color参数为颜色，可选intensity参数为强度
var pointLight = new THREE.PointLight('#ccffcc'); // 颜色可以是字符串、十六进制数值、或是THREE.Color对象
pointLight.position.set(-40, 60, -10);
pointLight.distance = 100;

点光源THREE.PointLight与聚光灯光源一样，都是使用摄像机来决定如何绘制阴影，所以也可以使用辅助类THREE.CameraHelper来展示场景中哪些部分被光源的摄像机所覆盖(阴影范围)，以及使用点光源的辅助类THREE.PointLightHelper来展示点光源的光线所照射的位置。

# 3.2.4 THREE.DirectionalLight（平行光光源）

该光源可以看做是距离很远的光，其发出的所有光线都是相互平行的（如太阳光，太阳距离地球非常远，到达地球的所有光线几乎都是平行的），且被该光源照亮的整个区域接收到的光强是一样的，不像聚光灯那样可以调节相关属性使得距离目标越远越暗淡。

平行光光源THREE.DirectionalLight与聚光灯光源有很多相同的的属性，如position、target、castShadow、shadow相关子属性(除camera.fov)、intensity、color。平行光光源也有对应的辅助类THREE.DirectionalLightHelper。

由于平行光光源的所有光线都是平行的，所以其不会像聚光灯光源那样照射范围是一个光锥，而是一个立方体区域，只有在该区域内的物体才可产生阴影。定义该区域类似第二章中规定正交投影相机OrthographicCamera的区域的几个参数：left、right、top、bottom、near、far，只是这几个属性都是放在平行光光源的shadow.camera属性上的。

var directionalLight = new THREE.DirectionalLight('#ccffcc');
directionalLight.castShow = true;
directionalLight.shadow.camera.near = 2;
directionalLight.shadow.camera.far = 80;
// ...left、right、top、bottom

# 3.3 特殊光源

# 3.3.1 THREE.HemisphereLight（半球光光源）

该光源可以创建出更加贴近自然的户外光照效果，因为在户外，并不是所有的光照都来自上方，很多是来自于大气的散射和地面以及其他物体的反射，而该半球光光源就是模拟这种环境效果的光源。

// new THREE.HemisphereLight(skyColor, groundColor, intensity); skyColor为天空颜色，groundColor为地面颜色，intensity为光源强度
var hemisphereLight = new THREE.HemisphereLight('#ccffcc', '#00ff00', 0.6);
hemisphereLight.position.set(0, 500, 0);

# 3.3.2 THREE.AreaLight（区域光光源/平面光光源）

使用该光源前还需要先引入RectAreaLightUniformsLib文件才能使用。该光源在最新版本的Three.js中已更改为THREE.RectAreaLight类，使用方法略微有所不同。

import 'threejs/src/lights/RectAreaLight.js';
// new THREE.RectAreaLight(color, intensity, width, height); color为光源颜色，intensity为光源强度，width/height为该发光平面的宽高
var rectAreaLight = new THREE.RectAreaLight('#ccffcc', 50, 4, 10); // 平面光光源的平面是垂直于地面的
rectAreaLight.position.set(0, 50, 0);
rectAreaLight.lookAt(0, 50, 0); // 可使用原型链上的lookAt()方法让光源朝向某位置，默认是朝向场景中心的

# 3.3.3 THREE.LensFlare（镜头光晕）

当把镜头直接朝向太阳或一个非常明亮的光源拍照时出现的效果就是光晕。创建镜头光晕的方式如下：

// 实例化光晕，new THREE.LensFlare(texture, size, distance, blending, color, opacity); 各参数说明见后面的表
var texture = THREE.ImageUtils.loadTexture('texture/image/path.png'); // 加载一个纹理图片，用于决定光晕的形状（主要位置）
var color = new THREE.Color('#ffaacc'); // 取一个颜色，也可以直接放在 THREE.LensFlare 的参数中(或使用其他颜色格式，如直接传颜色而不进行颜色实例化，因此时是作为参数传入，其内部会创建一个THREE.Color对象)
var lensFlare = new THREE.LensFlare(texture, 350, 0, THREE.AdditiveBlending, color);
var texture1 = THREE.ImageUtils.loadTexture('texture/image/path1.png'); // 加载其他纹理图片，用于添加光晕使得靠近镜头之间的画面的失真模糊效果(就像我们看很亮的光源时刺眼周围不是很清晰的现象)
// 使用LensFlare实例上的add()方法在主光晕的基础上添加额外的光晕，这些光晕就分布在光源到镜头之间
lensFlare.add(texture1, 60, 0.6, THREE.AdditiveBlending); // add()方法可接受的参数跟实例化时一样
lensFlare.add(texture1, 100, 0.8, THREE.AdditiveBlending);
lensFlare.add(texture1, 60, 1, THREE.AdditiveBlending);
