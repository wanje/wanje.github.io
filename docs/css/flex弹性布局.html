# flex弹性布局

flex: none | auto | [ <flex-grow> <flex-shrink>? || <flex-basis> ]；flex-grow表示有多余空间时对多余空间的分配比例，默认值0；flex-shrink表示空间不够时需要压缩的空间分配比例，默认值1； flex-basis表示元素的基本固定宽度，默认值auto(但Chrome较新版本中默认是0)，必须是一个表示尺寸的值。

  • flex: none，此时同flex: 0 0 auto，即flex-grow: 0、flex-shrink: 0、flex-basis: auto；
  • flex: auto，此时同flex: 1 1 auto，即flex-grow: 1、flex-shrink: 1、flex-basis: auto；
  • flex: 2，只有一个数值时，该值表示flex-grow，其余两个值使用默认值，也即等同于flex: 2 1 auto，但Chrome较新版本中等同于flex: 2 1 0；
  • flex: 100px，只有一个尺寸值时，该值表示flex-basis，其余两个值使用默认值，也即等同于flex: 0 1 100px，但Chrome较新版本中等同于flex: 1 1 100px；
  • flex: 2 2，两个数值时，该值表示flex-grow flex-shrink，而flex-basis使用默认值auto，也即等同于flex: 2 2 auto，但Chrome较新版本中等同于flex: 2 2 0；
  • flex: 2 100px，两个值且第二个为尺寸值时，该值表示flex-grow flex-basis，而flex-shrink使用默认值1，也即等同于flex: 2 1 100px；
  • flex: 1 2 100px，三个值时，该值就表示flex-grow flex-shrink flex-basis。
