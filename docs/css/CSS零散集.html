# min-content/max-content/fit-content

属性值 宽度计算方式 典型场景
min-content 内容中最长不可分割部分的宽度（最紧凑） 希望元素尽可能窄，比如表格列宽调整
max-content 内容不换行的完整宽度（最宽松） 希望元素完全展开，比如自适应内容宽度
fit-content 在 min-content 和 max-content 之间，取决于可用空间 内容自适应，但受容器约束，防止溢出

直观理解：

假设元素内容是 "Hello World"，容器宽度为 200px：

  • min-content：宽度是 "World" 的宽度（最长单词，假设约 50px）。
  • max-content：宽度是 "Hello World" 的完整宽度（假设约 120px），可能溢出容器。
  • fit-content：
    • 如果容器宽度足够（>120px），宽度是 120px（等于 max-content）。
    • 如果容器宽度不足（比如 80px），宽度会尽量接近 80px，但不会小于 min-content（50px）。
