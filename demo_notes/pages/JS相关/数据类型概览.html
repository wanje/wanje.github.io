5种基本数据类型包含：Undefined、Null、Boolean、Number、String
    1种复杂数据类型Object(引用类型)，但该类型引申了几种其他类型
    对未声明的变量调用delete并不会导致错误(原则说使用未声明变量会导致报错)，但这样并无意义，注意在严格模式下仍会报错（delete的目的是删除对象属性，一般不用于删除变量）

    使用typeof操作符检测给定的数据的数据类型，返回以下字符串之一：
    "undefined"：若检测值未定义（包括未初始化和未声明，注意undefined关键字也是）
    "boolean"：若检测值是布尔值（true或false）
    "string"：若检测值是字符串
    "number"：若检测值是数字或NaN
    "object"：若检测值是对象或null，注意该处的对象包含数组，以及日期、正则、Math、JSON这些内置对象
    "function"：若检测值是函数
    typeof是操作符，不是函数，所以其后不需要括号，但用了括号也并不会出错

    null == undefined     //true，但不全等
    null != {}            //true
    NaN != NaN            //true

    /*=================================================================================================================*/
    /*==================================================== Boolean ====================================================*/
    /*=================================================================================================================*/
    Boolean类型的字面值只有true和false两个，注意其是区分大小写的，都是小写，其他任何写法都不属于布尔值，只是标识符(即看作是变量名，如True)
    任何数据类型都可转换为对应的布尔值，显式方法有一个：Boolean()，转换规则如下：
    ------------|---------------------------------|-------------------
    数据类型     |    转换为true的值                 |   转换为false的值
    ------------|---------------------------------|-------------------
    Boolean     |    true                         |    false
    ------------|---------------------------------|-------------------
    String      |    任何非空字符(即使也有一个空格)   |    ""（空字符串）
    ------------|---------------------------------|-------------------
    Number      |    任何非零数值(包括无穷大)        |    0和NaN
    ------------|---------------------------------|-------------------
    Object      |    任何对象(即使是{})             |    null
    ------------|---------------------------------|-------------------
    Undefined   |    不适用                        |    undefined
    ------------|---------------------------------|-------------------


    /*================================================================================================================*/
    /*==================================================== Number ====================================================*/
    /*================================================================================================================*/
    平时最常用到的数字字面值是十进制表示法，也可以是八进制(以0开头，如：076)，十六进制(以0x开头，如：0x1f)
    八进制字面量在严格模式下是无效的，会报错，一般不推荐使用八进制字面量，可以使用字符串形式；八进制中若数位值超过表示范围(0~7)将忽略前导0认为是十进制数，如：079，此时会认为是十进制79
    须注意：不管是八进制还是十六进制，在算数运算时都会自动转换为十进制进行计算和输出结果，如：017+0x17  => 输出38，为十进制
    Number表示的数值是有范围的，能表示的最小和最大数值分别在以下常量中：Number.MIN_VALUE、Number.MAX_VALUE
    如果结果超过以上能表示的最大最小值，将被自动转换为特殊的 Infinity 值，表示无穷大，加上负号则表示负无穷，此时就不能再参与下一次计算了，因为已计算不出来
    访问 Number.NEGATIVE_INFINITY 和 Number.POSITIVE_INFINITY 也可得到负和正 Infinity 值，可见这两个属性中分别保存的就是 -Infinity 和 Infinity
    判断一个数值是否在能表示的最大最小值之间，可以使用 isFinite(参数) 函数，传入检测值，若在 Number.MIN_VALUE 与 Number.MAX_VALUE 之间则返回true，否则false（也可理解为判断是否为“非无穷”）
    NaN即非数值(not a number)，用于表示原本要返回数值的操作数却未返回数值的情况，如减法运算中包含不能被转换为数值的操作数,注意0除以0返回NaN，正数除以0返回Infinity，负数除以0返回-Infinity
    isNaN(参数) 函数用于判断给定的值是否不是一个数值，注意是判断“不是数值”，而不是判断是数值，该函数会将参数进行隐式转换为数值，任何不能准换为数值的就会返回true，否则false，注意true和false可以被转换为1和0
    当isNaN()函数传入一个对象obj时，内部会先调用对象的 valueOf() 方法确定其返回值能否转换为数值，若不能，则会基于这个返回值再调用 toString() 方法，再测试最后的返回值能否转换为数值
    数值转换，显式方法有三个：Number()、parseInt()、parseFloat()，第一种可用于任何数据类型，后面两张专门用于将字符串转换为数值
    Number()函数的转换规则如下(同一元加+操作符的隐式转换相同)：
    ------------|----------------------------------------------------
    数据类型     |    转换关系
    ------------|----------------------------------------------------
    Boolean     |    true => 1，false => 0
    ------------|----------------------------------------------------
    Number      |    不转换，只是简单的传入和返回
    ------------|----------------------------------------------------
    null        |    0
    ------------|----------------------------------------------------
    Undefined   |    NaN
    ------------|----------------------------------------------------
    String      |    1、若字符串为纯有效数值的字符串形式(包括正负值、浮点数、十六进制格式)，则都转换为相同大小的十进制数(注意只有八进制前导0会被忽略而当做是十进制)；
                |    2、若是空字符串(不包含任何字符，空格也没有)，则转换为0；     3、除1和2中情况都转换为NaN
    ------------|----------------------------------------------------
    Object      |    先调用对象的valueOf()方法并按前面的规则转换返回值，若转换结果为NaN，则再调用对象的toString()方法再次按前面的规则转换返回的字符串
    ------------|----------------------------------------------------
    在上面对字符串进行数值转换中，Number()只能将本身完全是数值形式的字符串进行正确转换，而parseInt()和parseFloat()能将只要开头是有效数值的字符串(会忽略前面的所有空格)中的整数或浮点数提取出来，若不是则都返回NaN，所以与Number()不同空字符串也将返回NaN

    parseInt('s')能够识别出各种整数格式(即前面所说的进制，但解析结果都是返回等值十进制整数)，即字符串若以“0x”开头且其后跟数字字符则按十六进制解析，若“0”开头且后面跟数字字符则按八进制解析(es3中是如此，但es5中会忽略前导零而当做十进制，为此解析八进制应传入第二个参数)，
    parseInt('s', n)方法可以传入第二个参数n表示解析基数(即按多少进制解析)，此时即使没有前导标志也会按指定进制解析，如：parseInt('af',16)===parseInt('0xaf',16)===parseInt('0xaf')，parseInt('77',8)===parseInt('077',8)但不一定==parseInt('077')
    可搭配num.toString(n)方法实现各种进制的转换与解析（参见下面String中toString方法的部分说明）

    parseFloat('s')可提取字符串前面的浮点数，只有第一个小数点有效，之后的小数点无效，且其只能按十进制解析，所以没有第二个参数，若提取的只有整数或浮点数小数点后都是0，这其返回的也是整数

    /*==================================================== String ====================================================*/
    /*================================================================================================================*/
    字符串字面量中的转义字符包含（注意每个转义字符占用一个实际字符）：
    --------|-----------------------------------------------------------------
    \n      |    换行
    --------|-----------------------------------------------------------------
    \t      |    制表
    --------|-----------------------------------------------------------------
    \b      |    退格
    --------|-----------------------------------------------------------------
    \r      |    回车
    --------|-----------------------------------------------------------------
    \f      |    进纸
    --------|-----------------------------------------------------------------
    \\      |    斜杆\
    --------|-----------------------------------------------------------------
    \'      |    单引号'
    --------|-----------------------------------------------------------------
    \"      |    双引号"
    --------|-----------------------------------------------------------------
    \xnn    |    以十六进制代码nn表示的一个字符(n为0~f)，如：\x41表示"A"
    --------|-----------------------------------------------------------------
    \unnnn  |    以十六进制代码nnnn表示的一个Unicode字符(n为0~f)，如：\u03a3表示"∑"
    --------|-----------------------------------------------------------------
    字符串转换有两种函数/方法：1、调用值的toString()方法；2、使用转型函数String()

    val.toString()方法：几乎每个值都有一个toString()方法(除了null和undefined)，调用该方法即可获得值val的字符串形式；
    注意：一般情况下调用toString()方法是不传参的，但调用数字的toString()方法时可以传入一个数字参数表示基数(即将数字转换成多少进制的字符串形式，默认为10)，结合parseInt()带第二个参数使用可实现各种进制的转换与解析
    num.toString(n)中num可以是合法的任意进制的数字(一般为十或十六进制)，将按照指定的进制n转换为等值数值的字符串形式，如若：num=0xaf；则 num.toString(10) => '175'; num.toString(8) => '257'; num.toString(16) => 'af';

    String()方法：传入要转换的值val
    若值val有toString()方法，则调用该方法(没有参数)并返回相应的结果；
    若值val为null，则返回"null"；
    若值val为undefined，则返回"undefined"

    除了调用上面的函数方法进行转换，要把某个值转换为字符串，还可以使用加号+操作符将其与一个字符串("")相加

    /*==================================================== Object ====================================================*/
    /*================================================================================================================*/
    使用new+构造函数方式创建对象实例时，若不需要给构造函数传入参数则也是可以省略后面的括号的(但不推荐这样做)，如：new Object 等同于 new Object()

    Object的每个实例都具有以下属性和方法：
    1、constructor：保存着用于创建当前对象的函数（构造函数）；
    2、hasOwnProperty(属性名)：检查给定的属性在当前对象实例中(不是实例的原型中)是否存在，属性名必须以字符串形式传入；
    3、isPrototypeOf(obj)：检查传入的对象是否是当前对象的原型；
    4、propertyIsEnumerable(属性名)：检查给定的属性是否能使用for-in语句进行枚举，属性名必须以字符串形式传入；
    5、toLocaleString()：返回对象的字符串表示，该字符串与执行环境的地区相对应；
    6、toString()：返回对象的字符串表示；
    7、valueOf()：返回对象的字符串、数值或布尔值表示，通常与toString()方法的返回值相同。
