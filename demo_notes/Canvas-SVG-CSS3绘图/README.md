# SVG 奇思妙想 
使用 `SVG` 标签或是CSS属性 `clip-path` 创建的图形

| Demo link | Description |
| ------| ------ | 
| [SVG奇思妙想](http://sbco.cc/demo/svg/html/index.html)  | 使用 `SVG` or `clip-path` 创建的图形 | 
| [CSS奇思妙想](http://sbco.cc/magicCss/html/index.html) | CSS3奇思妙想，单标签实现各类图形 |
