是的 自定义标签
  • webpack模板的项目包时 npm run dev 因webpack版本不同导致一些文件已替换npm install 再执行 npm run dev 时即成功，新版不会自动打开浏览器，可在config/index.js中将 autoOpenBrowser 修改为 true
  • 注意：指令和插值表达式中没加引号的都被认为是变量(都被认为是vue实例下的属性或方法， 加了引号才表示字符串，数组单元格和对象属性值也一样，只要是显式写在行内的都遵循此规则
  • v-cloak指令用于隐藏数据渲染完成前的双花括号，可在css中设置 [v-cloak] {display: none;}样式
  • 在模块化开发中用到样式scoped限制作用域时，有时效果无法深入到子元素中(尤其在修改第三方样式时)
