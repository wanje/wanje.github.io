材质就像物体的皮肤，决定了几何体的外表质感(注意与纹理区分，纹理就像套在皮肤外的衣服，或皮肤上的纹身)。带前缀Mesh的代表属于网格类型材质。

除上面表中的这些材质外，Three.js库源码中还有些其他材质，如用来给单个点设置样式的THREE.SpriteMaterial材质和THREE.PointMaterial材质，以及只能和THREE.BufferedGeometry几何体一起使用的THREE.RawShaderMaterial材质等。

# 4.1 理解材质的共有属性

Three,js中的材质都是基于材质基类THREE.Material扩展的，该类提供了所有的共有属性，这些属性分为：基础属性、融合属性、高级属性。该章节会跳过与纹理和贴图相关的属性(第10章介绍)，与动画相关的特殊属性(skinning、morphNormals、morphTargets)(第9章介绍)，及与剪切相关的clipIntersection、clippingPlanes、clipShadow3个属性(第6章介绍)。

# 4.1.1 基础属性

基础属性是最常用的，通过这些属性可以控制物体的透明度、是否可见、如何被引用(通过ID或是自定义名称)等。THREE.Material类的基础属性如下：

  • 基础属性中side属性表示哪一面应用材质，其有三个值：FrontSide(正面/外侧，默认)、BackSide(背面/内侧)、DoubleSide(正反两面/内外两侧)，这三个值实际对应的就是0、1、2，所以也可以直接用数字。这里的正面/背面/内侧/外侧是针对不同表现的几何体的，对于曲面或平面就是分正反面，而对于形成空间包围的立体图形来说就是分内外侧。

# 4.2 从简单的网格材质开始

传入属性配置材质的方式有两种，一种是在实例化时通过参数对象的方式传入参数，另一种是先创建实例，然后再分别设置相关属性。当然，两种方式也可以混合使用，先按方式一用对象参数进行实例化，再按方式二添加其他属性或覆盖属性值。一般来说按照方式一直接传入参数对象最好。

// 方式一：实例化时传入对象参数配置各属性
var material1 = new THREE.MeshBasicMaterial({
  color: '#eee',
  name: 'material-1',
  opacity: 0.7
  ...
});

// 方式二：先实例化，再在实例上分别设置需要的属性
var material2 = new THREE.MeshBasicMaterial();
material2.color = new THREE.Color('#eee'); // 通过属性赋值颜色时必须要使用'THREE.Color'对象，只有在作为相关类的参数传入时可以直接使用颜色字符串或数值
material2.name = 'material-2'; // 通过属性赋值颜色时必须要使用'THREE.Color'对象，只有在作为相关类的参数传入时可以直接使用颜色字符串或数值
material2.opacity = 0.7; // 通过属性赋值颜色时必须要使用'THREE.Color'对象，只有在作为相关类的参数传入时可以直接使用颜色字符串或数值

// 混合使用：方式一实例化后，按方式二增加设置新的属性或覆盖原来属性的值
material1.transparent = true;
material1.opacity = 0.77;

# 4.2.1 THREE.MeshBasicMaterial（网格基础材质）

该材质不考虑场景中光照的影响，使用该材质的网格会渲染成简单的平面多边形(因为不受光照影响，所有位置颜色都一样，也就看不出立体棱角)，也可以显示几何体的线框。除公用属性外，该材质还可以设置如下的一些属性： 

var meshBasicMaterial = new THREE.MeshBasicMaterial({color: '#eee'});
meshBasicMaterial.side = THREE.DoubleSide; // 定义几何体的哪一面应用材质，三个值：FrontSide(正面，默认)、BackSide(背面)、DoubleSide(正反两面)，这三个值实际对应的就是0、1、2，所以也可以直接用数字赋值

# 4.2.2 THREE.MeshDepthMaterial（网格深度材质）

使用该材质的物体，其外观不由光照或某个材质属性决定，而是由物体到摄像机的距离决定(越远获得的光线越暗颜色越淡)，跟其他材质结合使用可以很容易创建出逐渐消失的效果。结合该属性，通过控制相机的near和far属性，我们可以控制物体(颜色)消失的速度。因为near和far区域外的物体本就不会显示，而这两个属性又决定了相机的可视距离，而该距离就影响了MeshDepthMaterial材质物体的消失程度，距离越短，则near到far消失的越快，否则越慢。该材质只有额外两个控制线框显示的属性： 

var MeshDepthMaterial = new THREE.MeshDepthMaterial(); // 注意该材质没有 color 属性，默认只是白色。(可以使用联合材质，在同一物体上同时使用多个材质，以丰富物体表现)
scene.overrideMaterial = meshBasicMaterial; // 这里使用了场景scene的“overrideMaterial”属性，其将覆盖场景内所有对象的材质，强制所有对象使用该材质

# 4.2.3 联合材质（同时使用多种材质[作用在物体整体上]）

通过前面的MeshDepthMaterial材质可见其没有颜色设置，基本都是默认属性表现，要想物体有更好的外观表现，我们就可以使用联合材质，在同一物体上同时使用多种材质(此时就要用到融合属性了)。

var cubeGeometry = new THREE.BoxGeometry(4, 4, 4); // 创建一个4x4x4的立方体
var meshBasicMaterial = new THREE.MeshBasicMaterial({
  color: 0x00ff00,
  transparent: true, // 不设置为`true`就只会得到纯色物体，反应不出光线效果？实测似乎没有影响
  blending: THREE.MultiplyBlending // 融合模式设置，决定该材质如何与背景(这里就指MeshDepthMaterial材质渲染的cube方块)相互作用。此处THREE.MultiplyBlending模式对应“4”，该模式会将前景色与背景色相乘后输出
});
var meshDepthMaterial = new THREE.MeshDepthMaterial();
 // 下面的方法创建多材质网格对象，其中的几何体会被复制，有几种材质就会被复制成几个几何体，最后返回的是一个网格组，相当于几个应用不同材质而形状相同的几何体在同一位置重合在一起所表现出的样式
var cube = new THREE.SceneUtils.createMultiMaterialObject(cubeGeometry, [meshBasicMaterial, meshDepthMaterial]);
// 注意：网格组仍是一个整体的对象，只是其包含几个子对象(children中)，并不是其本身成了一个数组
cube.children[1].scale.set(0.99, 0.99, 0.99); // 将“MeshDepthMaterial”材质的网格缩小一点，避免渲染时画面闪烁(当渲染的物体有一个在别的物体上，且有一个物体是透明的时，闪烁现象时有发生，但并不是一定会出现，只是以防万一)

# 4.2.4 THREE.MeshNormalMaterial（网格法向材质）

应用该材质的物体表面的颜色由物体表面的法向量决定(也就是物体每个片元的法向量决定该片元的颜色)，所以法向量相同的位置的颜色始终是一样的。可以使用箭头辅助类THREE.ArrowHelper来显示每个片元上的法向量指向。

var meshNormalMaterial = new THREE.MeshNormalMaterial();
// 创建一个球体并使用法向材质
var sphereGeometry = new THREE.SphereGeometry(14, 20, 20);
var sphere = new THREE.Mesh(sphereGeometry, meshNormalMaterial);
// 添加法向量显示
var vertices = sphere.geometry.vertices; // 球体的所有顶点
sphere.geometry.faces.forEach(face => { // 计算每个片元的法向量
  var centroid = new THREE.Vector3(0, 0, 0); // 创建一个零向量
  centroid.add(vertices[face.a]); // 将每个片元的三个顶点向量相加(计算片元质心[也是几何中心])，add()方法改变的是当前向量
  centroid.add(vertices[face.b]);
  centroid.add(vertices[face.c]);
  centroid.devideScalar(3); // 向量的devideScalar()方法用于将一个向量除以一个标量(也就是缩短长度)，这里除以3是由于3个顶点距离特别近，相加的向量长度接近片元中心向量长度的3倍
  var arrow = new THREE.ArrowHelper(face.normal, centroid, 2, 0x3333ff, 0.5, 0.5); // 创建箭头
  sphere.add(arrow);
});

同网格深度材质MeshDepthMaterial一样，该材质也没有颜色设置，也只有与线框有关的wireframe和wireframeLinewidth两个额外属性。

THREE.ArrowHelper箭头辅助类的使用：

/** 
 * direction：箭头方向，一个方向向量，必须是单位方向向量
 * origin：箭头起点坐标(也是一个向量)
 * length：箭头长度，默认1
 * color：箭头颜色，注意只能是十六进制数值格式，默认0xffff00
 * headLength：箭头头部长度，默认为箭头长度的0.2倍(0.2 * length)
 * headWidth：箭头头部宽度，默认为箭头头部长度的0.2倍(0.2 * headLength).
 */
var arrow = new THREE.ArrowHelper(direction, origin, length, color, headLength, headWidth);

每个Mesh对象都可以看做是一个组(group，并不是数组)，可以使用其add()方法往其中添加额外的几何体，整个组可以看做是一个整体，add添加的对象就是其子对象，其children属性保存了所有子对象。

var group = new THREE.Mesh(); // 每个Mesh对象也是一个组，像上面的立方体cube、球体sphere也都是一个组
group.add(geometry1);
group.add(geometry2);
...

# 4.2.5 在单几何体上使用多种材质（每个面/片元使用不同的材质）

// 在网格化时，材质参数使用数组传入多个材质
var multiMaterialGeom = new THREE.Mesh(cubeGeometry, [material1, material2, ...]);

与联合材质THREE.SceneUtils.createMultiMaterialObject用法一样，但是作用不一样。联合材质中每种材质都是作用在物体整体上的，而此处是物体的不同面或片元按顺序使用材质组中的材质。几何体的每个面(几何体的faces属性存储了该几何体的所以面/片元)都具有一个materialIndex属性，该属性是该面所使用的材质在材质组中的索引，可以查看该值以确认每个面使用的材质组中的哪个材质。

该材质可在物体上创建一种光亮的效果（在物体表面实现高光效果），其既可以模拟塑料质感，也可以一定程度上模拟金属质感。该材质不仅包含暗淡材质THREE.MeshLambertMaterial的所有属性，还多额外两个属性： Three.js还提供了一个THREE.MeshPhongMaterial材质的扩展材质：卡通材质THREE.MeshToonMaterial，该材质的属性与MeshPhongMaterial完全相同，但是会以不同的方式渲染物体，看着更加卡通化。

# 4.4 线性几何体的材质

该部分要介绍的材质只能用于创建THREE.Line线段网格(作用同THREE.Mesh)，线段只由顶点组成，不包含任何面。Three.js库提供了两种可用于线段的材质：

  • THREE.LineBasicMaterial：基本线条材质，可设置color、linewidth、linecap、linejoin 和 vertexColors 属性。
  • THREE.LineDashedMaterial：虚线材质，该材质也具备基本线条材质的所有属性，还可以通过dashSize、gapSize 和 scale 属性指定虚线和空白间隙的长度来创建出虚线效果。

# 4.4.1 THREE.LineBasicMaterial（基本线条材质）

该材质有如下一些属性： 实际上vertexColors属性设置为一个真值就可以，THREE.VertexColors实际也是一个常量2。

var points = gosper(4, 60); // 这里假设'gosper()'是一个定义好的可以生成一组在一个平面上的点坐标的方法
var lines = new THREE.Geometry();
var colors = [];
var i = 0;
points.forEach(function(point) {
  lines.vertices.push(new THREE.Vector3(point.x, p oint.y, point.z)); // 几何体的vertices属性保存了所有顶点
  colors[i] = new THREE.Color(0xffffff);
  colors[i].setHSL(point.x/100+0.5, point.y*20/300, 0.8); // 这里是使用HSL格式颜色值，目的是更容易创建出匹配的颜色集合
  i++;
});
lines.colors = colors; // 顶点几何体的 colors 队列，与顶点数量和顺序保持一致
var lineBasicMaterial = new THREE.LineBasicMaterial({
  vertexColors: THREE.VertexColors // 设置为一个真值的效果都一样
});
var line = new THREE.Line(lines, lineBasicMaterial); // 这里THREE.Line与其他几何体的THREE.Mesh的作用一样，只不过其专用于线段

# 4.4.2 THREE.LineDashedMaterial（虚线材质）

该材质包含上面LineBasicMaterial材质的所有属性，还有如下可用来定义虚线宽度和间隙的额外属性：

  • scale属性缩放的实际是虚线的密度，默认为1，当值大于1时，密度增加，所以表现出来是线段和间隔变小了，而值小于1时，密度降低，从而线段和间隔增大。

该材质的用法与上面LineBasicMaterial材质基本一样，但最后要多一步调用computeLineDistances方法计算线段顶点间的距离。

var lineDashedMaterial = new THREE.LineDashedMaterial({
  vertexColors: true,
  dashSize: 10,
  gapSize: 1
});
var line = new THREE.Line(lines, lineBasicMaterial);
line.computeLineDistances(); // 不调用此方法则间隔将不会正确显示
