# 6.1 THREE.ConvexGeometry创建包围点云的最小图形(凸面)

该方法可以围绕一组点创建一个凸包，这个凸包就是包围这组点的最小图形，注意这里说的是包围，并不是边线要经过所有点(即这组点并不是全都会成为顶点，但顶点肯定属于这组点)。该方法属于three.js的扩展方法，需要额外引入examples/js/geometries/ConvexGeometry.js文件。

// 创建一组随机点
const points = Array(20).fill(1).map(() => {
  const x = -15 + Math.round(Math.random()*30);
  const y = -15 + Math.round(Math.random()*30);
  const z = -15 + Math.round(Math.random()*30);

  return new THREE.Vector3(x, y, z);  // 点用三维向量表示
});

// 创建包围给定点组的最小图形（传入点集数组）
const convexGeometry = new THREE.ConvexGeometry(points);
// 若要使图形渲染相对比较平滑(减少褶皱感)，则需要手动调用下面的法线计算方法（其他大部分几何体在创建对象时会自动计算这些法线，无需手动调用）
convexGeometry.computeVertexNormals();  // 计算顶点法线
convexGeometry.computeFaceNormals();  // 计算面法线
convexGeometry.normalsNeedUpdate = true;

# 6.2 THREE.LatheGeometry由曲线绕中心轴旋转得到图形

该方法允许由一条光滑的曲线(可称为样条曲线)围绕中心Z轴旋绕而得到一个图形，因是围绕Z轴旋转得到，所以曲线上的每个点都绕成一个圆(或部分圆弧)，得到类似花瓶这样的瓶罐柱体(或一部分柱体)。

const latheGeometry = new THREE.LatheGeometry(points);  // 根据点组构成的曲线创建几何体
// 所有未网格化的几何体都只是定义形状，还需要应用材质进行网格化后再添加到场景中才能看到
const latheMesh = new THREE.Mesh(latheGeometry, new THREE.MeshNormalMaterial({color: '#f00'}));

new THREE.ExtrudeGeometry(shape, options);

# 6.4 THREE.ParametricGeometry根据给定函数确定顶点绘制几何体

该方法主要就是自定义一个顶点坐标计算公式，从而绘制出各种几何体。

// 定义所有顶点计算方式，u、v取值区间为[0,1]，会根据ParametricGeometry中设置的分割段数自动进行取值组合依次调用以用于各顶点坐标计算，则分段数越多则生成的顶点越多，曲面越平滑（当然计算量也越大）
const radialWave = function (u, v, optionalTarget) {
  var result = optionalTarget || new THREE.Vector3();
  var r = 50;
  // 自定义顶点坐标，采用不同的公式就会得到不同的坐标组合，而形成各种几何体
  var x = Math.sin(u) * r;
  var z = Math.sin(v / 2) * 2 * r;
  var y = (Math.sin(u * 4 * Math.PI) + Math.cos(v * 2 * Math.PI)) * 2.8;

  return result.set( x, y, z ); // 返回一个顶点
};
// 根据上面的顶点可以创建出一个波浪形曲面
new THREE.ParametricGeometry(radialWave, 70, 70); // 其第一个参数为分配顶点的函数

其参数说明如下： 上面function中u、v取值区间为[0,1]，会分别按照slices、stacks设置的段数进行分割组合并调用该函数以计算每个顶点的位置，即slices和stacks决定了顶点计算函数function的调用频率，总共调用次数是slices * stacks次，即u和v的每个取值都会与对方的所有取值进行一次组合计算。examples/js/ParametricGeometries.js文件中可以查看更多几何图形的顶点坐标计算公式，如克莱因瓶、平面、二维莫比乌斯带、三维莫比乌斯带、管、环状扭结、球体。

# 6.5.1 THREE.TextGeometry渲染文本

文本渲染需要指定想用的字体，以及使用THREE.TextGeometry文本几何体进行拉伸处理，这里的拉伸配置项大多与前面提到的拉伸几何体相同，只有部分是特有的。注意这里说的的三维文本，若是要渲染二维文本并将其用作材质的纹理，那么就不应该使用这里的THREE.TextGeometry，因三维文本的JS开销较大，二维文本直接使用H5的canvas画布会更好(ctx.font设置字体，ctx.fillText绘制文本)，再将canvas画布作为纹理输入。

let loadedFont;
const fontload = new THREE.FontLoader();
// 不能直接使用字体源文件格式，必须转换为json格式或者js文件格式（见下面添加自定义字体）
fontload.load('./path/to/font/file.json', (res) => {
  loadedFont = res; // res 响应为创建好的字体对象
  render(); // 这里render方法是我们最初封装的渲染函数，用于重绘场景
});
new THREE.TextGeometry('文本内容', {  // 第二个参数为一些配置项
  font: loadedFont, // 必选项

# 6.6 使用二元操作组合网格

通过Three.js的扩展库ThreeBSP (opens new window)，我们可以将多个标准几何体组合在一起(相交、相减、联合等)创建出新的几何体。由于该库使用CoffeeScript开发，要使用的话要么需要再引入coffee-script.js运行时实施编译，要么需要将其预先编译为JS文件就可直接使用(推荐采用该方式，可以在本书示例代码中的找到编译后的JS文件libs/other/ThreeBSP.js)。该库提供了如下三个能力函数(实例方法)： 注意：这三个函数在计算时使用的都是网格的绝对位置，所以在应用这三个函数前，若将多个网格组合在一起(指成组)或使用多种材质，可能会得到一些奇怪的结果，故为不出现这些意料之外的情况应当确保使用的是未经组合的网格。

这三个实例方法是可以同时使用的，如先取图形1与图形2的相交部分，再取该相交部分除去与图形3相交的部分。

# 6.6.1 subtract相减

在当前对象上减去与指定对象相交的部分。

// 将相关几何图形抓换为`ThreeBSP`对象
const g1BSP = new ThreeBSP(geometry1);
const g2BSP = new ThreeBSP(geometry2);
const resultBSP = g1BSP.subtract(g2BSP);  // 表示在`g1BSP`上减去其与`g2BSP`相交的部分，结果是`g1BSP`剩余的部分，得到的`resultBSP`结果可以继续与其他几何体进行类似操作
const result = resultBSP.toMesh();  // 将融合结果转换为mesh网格
// 手动调用新几何体的法线计算，使新几何体相对平滑
result.geometry.computeFaceNormals();
result.geometry.computeVertexNormals();

# 6.6.2 intersect相交

获取当前对象上与指定对象相交的部分。

// g1BSP、g2BSP同上
const resultBSP = g1BSP.intersect(g2BSP);  // 表示在`g1BSP`上只取其与`g2BSP`相交的部分
const result = resultBSP.toMesh();  // 将融合结果转换为mesh网格
// 手动调用新几何体的法线计算，使新几何体相对平滑
result.geometry.computeFaceNormals();
result.geometry.computeVertexNormals();
